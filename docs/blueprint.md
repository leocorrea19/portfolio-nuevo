## Style Guidelines:

- Primary color: Midnight blue (#191970) to convey professionalism and trust.
- Background color: Light gray (#F0F0F0) for a clean and modern feel.
- Accent color: Electric blue (#7DF9FF) to highlight key elements and call-to-action buttons.
- Headline font: 'Poppins', a geometric sans-serif, to present a modern and fashionable image; Body font: 'PT Sans', a humanist sans-serif, for easy readability of the text.
- Use professional and clean icons for skills and services sections.
- A clean and well-organized layout with clear sections for easy navigation. The design will incorporate the sketched layout provided.
- Subtle transitions and hover effects to enhance user experience without being distracting.
